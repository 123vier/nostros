# Getting Started

## Required Libraries and Framworks
- Node 16.x
- JDK 17.0.x
- (Android) Android Studio SKD
- yarn

# Running

- Open a virtual device
- Install
```
yarn format
```
- Run Metro
```
yarn start
```
